Senior Backend Engineer with many years of experience in Java, currently focused on Go.

## Current focus

This GitHub is focused on hands-on study in backend and platform engineering.

Current topics:

- Go
- gRPC and Protobuf
- Event-driven architecture
- RabbitMQ
- Redis and MongoDB
- Kubernetes
- Terraform and Infrastructure as Code
- Service mesh with Istio
- Observability with OpenTelemetry, Prometheus, and Grafana

To support that study, I’m building **Elaris**, a distributed resort simulation platform fully written in **Go** to explore service communication, asynchronous workflows, infrastructure, and runtime visibility.
